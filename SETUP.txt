================================================================================
🔧 K-pop 프로젝트 첫 실행 시 필수 설정 (SETUP GUIDE)
================================================================================

📋 단계별 설정 가이드

================================================================================
1단계: 프로젝트 다운로드
================================================================================

https://github.com/ymi-hub/K-pop.git 에서 클론하거나 다운로드
폴더명: K-pop


================================================================================
2단계: VS Code에서 K-pop 폴더 열기
================================================================================

File → Open Folder → K-pop 폴더 선택


================================================================================
3단계: 터미널에서 의존성 설치 (5-10분 소요)
================================================================================

npm install


================================================================================
4단계: 환경 파일 생성
================================================================================

cp .env.example .env


================================================================================
5단계: Spotify Client ID 설정
================================================================================

.env 파일을 열어서 아래를 입력:

EXPO_PUBLIC_SPOTIFY_CLIENT_ID=your_spotify_client_id_here

**Spotify Client ID 얻는 방법:**
1. https://developer.spotify.com/dashboard 접속 (계정 로그인)
2. "Create an App" 클릭
3. 앱 생성 후 "Client ID" 복사
4. 위의 your_spotify_client_id_here 부분에 붙여넣기


================================================================================
6단계: Spotify Redirect URI 설정
================================================================================

1. https://developer.spotify.com/dashboard 접속
2. Your App 클릭
3. "Edit Settings" 클릭
4. Redirect URIs 섹션에 다음 추가:

http://localhost:19006/
http://localhost:3000/
http://localhost:8081/

5. Save 클릭


================================================================================
7단계: 앱 실행
================================================================================

**웹 버전 (권장):**
npm run web

**iOS 에뮬레이터:**
npm run ios

**Android 에뮬레이터:**
npm run android


================================================================================
필수 요구사항 체크리스트
================================================================================

[ ] Node.js 18+ 설치됨
[ ] npm 설치됨
[ ] Expo CLI 설치됨: npm install -g @expo/cli
[ ] Spotify 계정 생성함 (무료)
[ ] Spotify Developer 앱 생성함
[ ] Client ID 복사함
[ ] .env 파일에 Client ID 입력함
[ ] Redirect URIs 설정함


================================================================================
문제 해결
================================================================================

**"npm install" 실패 시:**

rm -rf node_modules package-lock.json
npm cache clean --force
npm install


**"Spotify Client ID Invalid" 에러 시:**
- .env 파일에 올바른 Client ID가 입력되었는지 확인
- Spotify Developer Dashboard에서 Client ID 다시 복사해서 붙여넣기


**"redirect_uri: Not matching configuration" 에러 시:**
- Spotify Developer Dashboard의 Edit Settings에서 Redirect URIs 재확인
- http://localhost:19006/ 반드시 포함되어 있는지 확인
